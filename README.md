# Shopping Mall Frontend

A modern, responsive shopping mall application built with React, TypeScript, and Material-UI.

## Features

- 🛍️ Product listing with responsive grid layout
- 🔍 Real-time search functionality
- 🎛️ Advanced filtering (price, category, brand, rating)
- 📱 Mobile-responsive design
- 🛒 Shopping cart integration ready
- ⭐ Product ratings and reviews
- 📄 Pagination for large product sets

## Tech Stack

- **React 18** - Modern React with hooks
- **TypeScript** - Type-safe development
- **Material-UI (MUI)** - Component library and design system
- **Vite** - Fast build tool and dev server

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
├── components/          # Reusable UI components
│   ├── common/         # Common components (Header, Footer)
│   └── ui/             # UI components (ProductCard, FilterSidebar)
├── features/           # Feature-specific components
│   └── products/       # Product-related features
├── hooks/              # Custom React hooks
├── services/           # API services
├── store/              # State management
├── styles/             # Global styles
└── utils/              # Utility functions
```
