# Admin Dashboard Frontend

## Project Overview
This is a React TypeScript admin dashboard application built with Vite and Material-UI (MUI).

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## Project Structure
```
src/
├── assets/         # Static assets like images, fonts
├── components/     # Reusable, shared components
│   ├── common/     # Common components
│   ├── ui/         # UI components
│   └── layout/     # Layout components
├── features/       # Feature-specific modules
│   └── dashboard/  # Dashboard feature
├── hooks/          # Custom hooks
├── services/       # API clients, external services
├── store/          # State management
├── styles/         # Global styles
└── utils/          # Utility functions
```

## Technologies Used
- React 18
- TypeScript
- Vite
- Material-UI (MUI)
- Emotion (CSS-in-JS)
```
